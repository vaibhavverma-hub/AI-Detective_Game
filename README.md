🕵️ AI Detective Game

An AI-powered murder mystery game built using Python, Streamlit, and OpenAI.

Overview

AI Detective Game is an interactive detective experience where players investigate murders, collect clues, identify contradictions, and accuse suspects.

The game combines classic detective gameplay with AI-generated mysteries to create unique investigations.

---

Features

Crime Investigation

- Dynamic crime scenes
- Evidence collection
- Clue discovery system

Suspect System

- Multiple suspects
- Individual alibis
- Hidden motives
- Contradictions between statements and evidence

Detective Progression

- Detective score system
- Detective rank progression
- Investigation tracking

User Interface

- Streamlit-based web application
- Detective-themed layout
- Sidebar detective panel
- Suspect cards
- Clue display

AI Integration

- OpenAI-powered case generation
- Dynamic victims
- Dynamic suspects
- Dynamic clues
- Dynamic contradictions

---

Tech Stack

- Python
- Streamlit
- OpenAI API
